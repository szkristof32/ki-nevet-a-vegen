# Ki nevet a végén

## How to build

To build it using Visual Studio just run the `Bootstrap.bat` file, and open the solution file. You should be good to go.

To build it on Windows using Make run the `Bootstrap_HSZK.bat`.

On Linux run the `Bootstrap.sh` file. If it isn't marked as executable first run `chmod +x Bootstrap.sh`. Then you can compile it using make.

## How to play

You can roll the dice with the right mouse button, and then select one of your pieces by clicking on them. They should perform the correct number of steps.

To pan the camera press and hold the middle mouse button, then move the mouse. You can zoom in and out using the mouse wheel.
